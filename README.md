# rb
View transactions from an Angular UI

## Running the backend:  
cd ~rb-backend  
npm install  
npm run start:windows  

## Running the frontend:  
cd ~rb-frontend  
npm install  
ng serve  
in your browser navigate to http://localhost:4200/
